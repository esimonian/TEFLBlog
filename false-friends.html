False Friends, Real Freinds, and Strangers

11 Sep 2016 on Lexical, Teaching

We Have Terms For These But Let’s Use Cutesy Nicknames

1) False Friends are basically false cognates. Sopa is soup and not soap in Spanish. My book also wants us to remember that deep down ALL words are false cognates. I don’t think that’s super helpful or true, although the Whorf-Sapir fangirl in me wants to. A table is a table is a table, really. Although we have different cultural associations, we can recognize a table.

Side Note Though

It IS true that when we are acquiring a second language, we don’t map the words to the concepts, as we did in first language acquisition. We map the word in L2 with the word in L1, which is mapped to the concept that we learned as a kid. Helpful in that we don’t have to re-acquaint ourselves with the concept of the word “dog”, but less helpful in that we have a harder time getting to fluency when we see a dog and have to think first “dog” and then “perro”.

2) Real Friends are cognates. I don’t know why we’re calling the words “friends” instead of cognates when this is a class for grown-ass adults, but whatever. There are generally more cognates than false cognates between languages.

3) Strangers are untranslatables. Although nothing is REALLY untranslateable, it is true that there are sometimes words for which there is no one-for-one translation. Like schadenfreude.
